# Исправление: AI Reminder marker не парсится

## Проблема

На скриншоте видно что AI-бот выдаёт сырой маркер `[[REMINDER:...]]` пользователю. Значит:
1. `parseReminderMarker` возвращает `null` — regex не матчит
2. `stripReminderMarkers` тоже не матчит — маркер остаётся в тексте

Причина: LLM (qwen) может оборачивать JSON в фигурные кавычки, добавлять пробелы/переносы внутри маркера, или использовать нестандартные кавычки.

## Исправление

### `server/app.js` — функция `parseReminderMarker`

Заменить regex на более устойчивый:

```js
function parseReminderMarker(text) {
    const cleaned = String(text || '');
    const match = cleaned.match(/\[\[REMINDER[:\s]*(\{[\s\S]*?\})\s*\]\]/);
    if (!match) return null;
    try {
        const raw = match[1].replace(/[""]/g, '"').replace(/['']/g, "'");
        const data = JSON.parse(raw);
        if (!data.message || !data.runAt) return null;
        const runAtDate = new Date(data.runAt);
        if (Number.isisNaN(runAtDate.getTime())) return null;
        return { message: String(data.message).trim(), runAt: runAtDate.toISOString() };
    } catch (e) {
        console.error('[AI Bot] Reminder parse failed:', e.message, 'raw:', match[1]);
        return null;
    }
}
```

### `server/app.js` — функция `stripReminderMarkers`

Заменить regex на более широкий:

```js
function stripReminderMarkers(text) {
    return String(text || '')
        .replace(/\[\[REMINDER[\s\S]*?\]\]/g, '')
        .replace(/\n{3,}/g, '\n\n')
        .trim();
}
```

### Ключевые отличия от текущего кода

| Было | Стало |
|------|-------|
| `\[\[REMINDER:(\{.*?\})\]\]/s` | `\[\[REMINDER[:\s]*(\{[\s\S]*?\})\s*\]\]` |
| `.*?` — не матчит `\n` даже с `/s` | `[\s\S]*?` — матчит всё включая переносы |
| `:` после REMINDER — жёстко | `[:\s]*` — допускает пробелы |
| Без замены кавычек | Замена `""` на `"` и `''` на `'` |
| strip: `\[\[REMINDER:\{.*?\}\]\]/gs` | strip: `\[\[REMINDER[\s\S]*?\]\]/g` |

## Верификация

1. Деплой
2. Отправить AI-боту: «напомни через 30 сек покурить»
3. Ожидаемый результат:
   - В консоли: `[AI Bot] Reminder marker check: FOUND`
   - В консоли: `[AI Bot] Create reminder result: SUCCESS`
   - Пользователь видит чистый ответ без `[[REMINDER:...]]`
   - Через 30 сек Reminder-бот доставляет напоминание
